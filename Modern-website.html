Linux Commands
  • Add Users:
  • sudo adduser username: Add a user.
  • sudo usermod -aG groupname username: Add a user to a group.
  • sudo adduser username1 username2 username3: Add multiple users at once.
  • sudo usermod -aG groupname username1 username2: Add multiple users to a group at once.
  • Delete Users:
  • sudo deluser username: Delete a user.
  • sudo deluser username1 username2: Delete multiple users at once.
  • sudo deluser username1 username2 -G groupname: Remove multiple users from a group.
  • Add a Group:
  • sudo addgroup groupname: Add a group.
  • List Users and Groups:
  • sudo getent passwd: List all users.
  • sudo getent group: List all groups.
  • sudo getent passwd | grep groupname: List all users in a specific group.
  • Change User Passwords:
  • sudo passwd username: Change a user's password.
  • sudo passwd -e username1 username2: Change multiple users' passwords.
  • Grant elevated user rights:
  • sudo usermod -aG sudo username: Add a user to the sudo group (granting sudo privileges).
